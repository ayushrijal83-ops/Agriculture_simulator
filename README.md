📖 About the Project

AgroVision Nepal is an AI-powered agriculture simulation platform developed to demonstrate how Artificial Intelligence and modern web technologies can improve farming and agricultural decision-making.

The application enables users to identify crops using AI, simulate crop growth, explore agricultural funding opportunities, and gain valuable farming insights through an intuitive web interface.

The project aims to bridge the gap between technology and agriculture while showcasing the potential of AI in modern farming.

🚀 Installation

Clone Repository

git clone https://github.com/ayushrijal83-ops/Agriculture_simulator.git

---

Enter Project

cd Agriculture_simulator

---

Create Virtual Environment

Windows

python -m venv venv

Activate

venv\Scripts\activate

Linux / macOS

python3 -m venv venv

source venv/bin/activate

---

Install Dependencies

pip install -r requirements.txt

---

Run Application

python app.py

Open your browser and visit:

http://127.0.0.1:5000
